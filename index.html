🤖 How Our AI Works

BioBase uses a rule-based NLP (Natural Language Processing) system for symptom-driven disease inference — a core AI technique:

1. Tokenisation
User query split into meaningful tokens using regex pattern matching.
2. KB Matching
Tokens matched against a symptom–disease ontology (knowledge base).
3. Scoring & Ranking
Diseases scored by match count and ranked by relevance for output.
4. Fuzzy Matching
"Did you mean?" powered by Python difflib sequence matching.
